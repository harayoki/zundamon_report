# OpenAI API の料金目安（2024年10月時点）

> ※公式の最新価格は https://openai.com/pricing を参照してください。ここでは 2024 年 10 月時点で一般的に案内されているトークン単価の目安をまとめています。

## GPT-4o 系
- **GPT-4o**: 入力 $5.00 / 100万トークン、出力 $15.00 / 100万トークン。
- **GPT-4o Mini**: 入力 $0.15 / 100万トークン、出力 $0.60 / 100万トークン。
- **GPT-4o Realtime**: ストリーミング前提のモデルで、入力 ~$0.10 / 100万トークン、出力 ~$0.40 / 100万トークン が目安（レイテンシ課金は別途）。

## GPT-4 Turbo / その他
- **GPT-4 Turbo** (旧名称 4.1 など): 入力 ~$10.00 / 100万トークン、出力 ~$30.00 / 100万トークン。
- **GPT-3.5 Turbo**: 入力 ~$0.50 / 100万トークン、出力 ~$1.50 / 100万トークン（軽量用途向け）。

## コスト試算の例
- 会議議事録（日本語 10,000 文字 ≒ ~15,000 トークン）を **GPT-4o** で校正した場合:
  - 入力: 15k トークン → 約 $0.075
  - 出力: 15k トークン → 約 $0.225
  - 合計: 約 $0.30 / 回
- 同じ処理を **GPT-4o Mini** に切り替えると、合計は約 $0.011 まで下がります。

## 運用上のポイント
- 音声→文字起こし（Whisper など）を含める場合、LLM と別に音声モデルの料金がかかる点に注意してください。
- 価格は変更されることがあります。特にプレビュー期間中のモデルは頻繁に改定されるため、定期的な見直しを推奨します。
- トークン数は言語・文字種・システムプロンプトの長さで大きく変動するため、バッチ処理前にサンプルで概算するのが安全です。

### モデルを切り替えて使う場合の課金は？

- **従量課金はモデルごとに「使った分だけ」発生します。** 同じ API キーで GPT-4o と GPT-4o Mini を混在させても、請求は「各モデルに対する使用トークン数 × そのモデルの単価」の合計になります（モデル単位で月額固定費が発生するわけではありません）。
- たとえば 1 つのバッチ処理で 8 割を GPT-4o Mini、2 割を GPT-4o に振り分けるワークロード構成でも、合計請求額は両モデルの従量課金の足し合わせです。
- Whisper など別種のモデルを併用する場合も、各 API で使った分だけがそれぞれ加算されます。
